# Desafio Técnico - Analista de Bioinformática - Dasa

Aplicação web para consulta de variantes genéticas humanas utilizando a API pública do Ensembl.

## Funcionalidades
- Consulta de variantes por rsID
- Integração com Ensembl REST API
- Retorno JSON padronizado
- Visualização gráfica de dados
- Interface web intuitiva

## Tecnologias
- Flask 3.0.0
- Python 3.9+
- Docker
- UV (gerenciador de pacotes)
- Ensembl REST API
